jQuery_78 - Manipular el DOM: Rodear un elemento: wrap, wrapAll, wrapInner, unwrap

wrap --> Rodea con otro elemento que puede ser un elemento o cadena html.
wrapAll --> Rodea los elementos seleccionados con otro elemento.
wrapInner --> Rodea todos los hijos (incluyendo texto) con otro elemento.
unwrap --> Elimina los padres de un elemento.
