Ejemplos OffsetHeight, OffsetWidth, OffsetLeft, OffsetRight y OffsetParent

OffsetHeight

offsetHight = height+padding+border
height: 200px; border: 1px; padding: 15px;


OffsetWidth

offsetWidth = width+padding+border
width: 300px; border: 5px; padding: 15px;


OffsetLeft

offsetLeft = left + margin + padding(elemento padre)
left:20px;padding(elemento padre):15px;


OffsetTop

offsetTop = top + margin + padding(elemento padre)
top:30px;padding(elemento padre):15px;


OffsetParent

Muestra el elemento padre de un elemento
